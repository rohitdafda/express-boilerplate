4. JOI
5. JWT
6. Multer
7. Winston and Morgan

# Features
1. Centralized Routing 
2. Centralized Error Handling
3. Validation using custom error messages with JOI (Includes Params & QueryString Validation Also)

5. File Upload using Multer
